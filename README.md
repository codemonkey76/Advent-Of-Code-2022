# Advent of Code 2022

This is my attempt at the advent of code 2022.

## How to run

### Step 1. Clone the Repo
```bash
git clone https://github.com/codemonkey76/Advent-Of-Code-2022.git advent
```

### Step 2. Install Dependencies
```bash
cd advent
npm install
```

### Step 3. Run the Project
```bash
npm run dev
```
